### 이분탐색
정렬된 배열을 반으로 나누어 가며 원하는 값을 탐색
```java
static int binarySearch(int target){
  int start = 0;
  int end = N-1;
  while(start <= end){
    int mid = (start+end)/2;
    if(A[mid] < target) start = mid+1;
    else if(A[mid] > target) end = mid-1;
    else return mid;
  }
  return -1;
}


int index = Arrays.binarySearch(A, target); //index값 return, 값이 없다면 음수
```
